Features

Type-Safe Commands

Full TypeScript inference from schema to handler. Arguments and flags are automatically typed.

Rich Error System

Actionable errors with hints and documentation links. Better DX for failures.

Task Runner API

Multi-step workflows with progress tracking. Perfect for deployments and migrations.

UI Components

Spinners, progress bars, tables, cards, trees, toasts, dashboards, and more.

Interactive Prompts

Built-in text, password, confirm, select, and multiselect prompts.

Colored Help

Beautiful, colorized help output generated automatically from your schema.

Quick Start

# Install
npm install easycli-core easycli-ui

# Create your CLI
import { defineCLI } from "easycli-core";

const cli = defineCLI({
  name: "myapp",
  version: "1.0.0",
  commands: {
    deploy: {
      args: { env: ["staging", "production"] },
      async run({ env }, ctx) {
        const t = ctx.task("Deploying");
        await t.step("Build", build);
        await t.step("Upload", upload);
        t.success("Done!");
      }
    }
  }
});

cli.run();
